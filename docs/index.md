# Welcome to Order Daemon

Order Daemon is a powerful automation tool for WooCommerce that saves you time by handling routine order-processing tasks after checkout. It allows you to create simple but powerful rules to manage orders automatically, so you can focus on growing your business.

## Core Concepts

Order Daemon's automation engine is built on three simple concepts:

*   **Triggers:** An event that starts a rule. This is typically when an order is created or a payment is received.
    *   *Example:* `Order is Processing`

*   **Conditions:** A set of "if" statements that check the details of an order to see if it matches your criteria.
    *   *Example:* `If Order Total is greater than $100`

*   **Actions:** The task that Order Daemon performs when an order matches the trigger and conditions.
    *   *Example:* `Complete the Order`

By combining these three elements, you can automate a wide range of post-purchase workflows without writing a single line of code.

## Who is Order Daemon For?

Order Daemon is designed for anyone who wants to make their WooCommerce store run more efficiently.

*   **Shop Owners:** If you find yourself manually completing orders, sorting through orders to find specific types, or wishing you could apply a "set it and forget it" process to your fulfillment, Order Daemon is for you. It gives you back time so you can focus on your products and customers.

*   **Agency & Freelance Developers:** When building sites for clients, you need to deliver value quickly. Order Daemon provides a robust, reliable foundation for automating order management, allowing you to implement custom logic that would otherwise require hours of custom coding and testing.

## What Can You Automate?

Even with the free version of Order Daemon, you can build powerful workflows. Here are a few examples to spark your imagination:

*   **Instantly complete orders** for all your digital and virtual products.
*   Create a rule to **handle specific product categories** differently, like holding all "Pre-Order" items for review.
*   **Isolate high-value orders** for manual fraud checks before they are processed.
*   Filter and act on orders containing **specific types of products**, like subscriptions or bundles.
