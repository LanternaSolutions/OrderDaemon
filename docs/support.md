# Support & Troubleshooting

If you encounter any issues or have questions, here are the best ways to get help.

## Self-Service Diagnostics

Order Daemon comes with a built-in diagnostics tool to help you identify and resolve common problems quickly.

*   **Location:** Navigate to **WooCommerce > Order Daemon > Diagnostics**.
*   **What it does:** The tool checks for common issues related to your server configuration, API connectivity, and plugin setup.

Before reaching out for support, please run the diagnostics tool. It can often pinpoint the source of the problem instantly.

## Contacting Support

If you're still experiencing issues after running the diagnostics tool, please contact us through our official support channel on the wordpress.org support forums.

When submitting a support request, please include:

*   A summary of the issue you're experiencing.
*   The results from your diagnostics check.
*   Any relevant details about the rule that is not working as expected.

Providing this information will help us resolve your issue as quickly as possible.
